# Biine - Vite + React + Firebase scaffold

This is a paste-ready scaffold for the Biine classifieds app.
- Vite + React
- Tailwind CSS
- Firebase v9 modular SDK
- Auth: Email, Google, Facebook
- Ads: post with up to 5 images (Firebase Storage)
- Firestore: ads collection

## Setup
1. Copy `.env.example` to `.env` and fill values.
2. npm install
3. npm run dev
